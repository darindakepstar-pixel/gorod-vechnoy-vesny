# Город вечной весны — прототип

Визуальная новелла. Работает в браузере на телефоне и на компьютере.

## Файлы

| Файл | Что это |
|---|---|
| `scenes.js` | **Весь текст игры.** Правишь только его, когда пишешь сюжет. |
| `engine.js` | Движок: печать текста, выборы, счётчики, сохранения. Трогать не нужно. |
| `index.html` | Разметка и оформление. |
| `config.js` | Ключи Supabase. Пока заглушки — игра работает локально. |
| `supabase.sql` | Миграция для облачных сохранений. |
| `check.js` | Проверка сюжета на битые ссылки. |

## Как запустить локально

Просто открой `index.html` двойным кликом. Всё.

## Как выложить для подруг

1. Создай репозиторий на GitHub, залей туда все файлы.
2. Settings → Pages → Source: `main`, папка `/ (root)`.
3. Через минуту ссылка вида `https://твойник.github.io/имя-репо/` — её и рассылай.
4. На телефоне «Добавить на главный экран» — откроется как приложение.

## Как включить аккаунты и общую статистику

1. Создай проект в Supabase.
2. SQL Editor → вставь содержимое `supabase.sql` → Run.
3. Settings → API → скопируй Project URL и ключ `anon public`.
4. Вставь их в `config.js`.

На титуле вместо «локальный режим» появится «облако подключено».
Статистику смотри в Supabase: Table Editor → вью `stats`.

## Как писать новые сцены

Открой `scenes.js`, скопируй любую сцену, поменяй `id`, текст и `next`.
Обычная сцена:

```js
ch3_01: { bg:'city', who:'ТАНК', sprite:{char:'tank', emo:'happy'},
  text:'Ты вообще ела сегодня?', next:'ch3_02' },
```

Сцена с выбором (тогда `next` не нужен):

```js
ch3_05: { bg:'city', text:'Он ждёт ответа.',
  choices:[
    { text:'Согласиться.', love:{tank:2}, next:'ch3_06a' },
    { text:'Отказаться.',  love:{tank:-1}, flags:['отказ'], next:'ch3_06b' },
  ]},
```

Доступные фоны и персонажи перечислены вверху `scenes.js`.
После правок запусти `node check.js` — покажет опечатки в ссылках между сценами.

## Отладка

Клавиша `D` на компьютере или кнопка в меню — показывает счётчики симпатии,
текущую сцену и накопленные флаги. Перед раздачей подругам просто не жми её.

## Управление

- Клик или тап по тексту — дальше. Во время печати — домотать реплику.
- Пробел / Enter на компьютере — то же самое.
- «Лог» — всё прочитанное. «Меню» — сохранения и скорость текста.

## Чего пока нет
